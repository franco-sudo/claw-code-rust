# 📦 Claw RS v0.2.0 发布说明

## ✅ 版本更新完成

**当前版本**: `v0.2.0`  
**编译时间**: 2026-04-02  
**安装状态**: ✅ 已安装到 `/usr/local/bin/claw`

---

## 🎯 v0.2.0 新特性

### 1. 全局命令支持
- ✅ `claw` 命令可在任何终端目录使用
- ✅ API Key 自动保存，一次配置永久使用
- ✅ 快速启动 GLM-4 模型

### 2. 交互模式增强 ⭐
- ✅ **完美支持中文输入**（UTF-8）
- ✅ **箭头键导航**（←→移动光标，↑↓查看历史）
- ✅ **命令历史持久化**（自动保存到 `~/.claw-code-rust/history`）
- ✅ **丰富的快捷键**（Ctrl-A/E/U/W 等）
- ✅ **专业级命令行编辑**（基于 rustyline 库）

### 3. Bug 修复
- ✅ 修复 UTF-8 字符串截断导致的 panic
- ✅ 修复智谱 API URL 拼接问题
- ✅ 优化环境变量配置逻辑

## 🚀 快速开始

### 检查版本
```bash
claw --version
# 输出：claw-rs 0.2.0
```

### 启动交互模式
```bash
claw
> 帮我写一个 Rust 函数  # 中文输入无压力
> ←→ 左右移动光标修改    # 精准定位
> ↑↓ 查看历史命令        # 命令历史
```

### 单次查询
```bash
claw -q "列出当前目录的文件"
```

## 📁 配置文件位置

### 1. API 配置
```
~/.claw-code-rust/config.json
```
包含：
- Provider 设置
- API Key（自动保存）
- Base URL
- 默认模型

### 2. 命令历史
```
~/.claw-code-rust/history
```
包含：
- 所有历史输入记录
- 自动保存和加载

## 💡 使用技巧

### 首次使用
```bash
# 设置 API Key（仅首次需要）
export ZHIPU_API_KEY='your_api_key'

# 启动 claw
claw

# 之后无需再次设置，API Key 已自动保存
```

### 快捷键速查
| 按键 | 功能 |
|------|------|
| ← → | 左右移动光标 |
| ↑ ↓ | 翻阅历史命令 |
| Ctrl-A | 跳到行首 |
| Ctrl-E | 跳到行尾 |
| Ctrl-U | 删除整行 |
| Ctrl-W | 删除单词 |
| Ctrl-C | 中断输入 |
| Ctrl-D | 退出程序 |

### 搜索历史
1. 输入几个字符
2. 按 ↑ 键
3. 自动匹配历史记录
