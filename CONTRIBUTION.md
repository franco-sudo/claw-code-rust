# Contribution Guide

Thanks for your interest in contributing.

This project is still in an early stage, so small and focused contributions are especially helpful.

## How to Contribute

- Open an issue for bugs, ideas, or design discussions
- Submit a pull request for docs, translations, or implementation improvements
- Keep changes small and easy to review
- Prefer consistency with the existing crate structure and documentation style

## Notes

- Please describe the motivation behind your change
- If the change is large, discuss it in an issue first
- By contributing, you agree that your work can be distributed under the [MIT License](./LICENSE)
