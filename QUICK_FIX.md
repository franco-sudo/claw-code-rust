## 🚀 立即体验
```bash
claw
> 帮我写一个 Rust 函数  # 中文输入无压力
> ←→ 左右箭头移动光标    # 精准定位
> ↑↓ 上下箭头查看历史    # 命令历史
```

## 💡 快捷键速查
| 按键 | 功能 |
|------|------|
| ← → | 左右移动光标 |
| ↑ ↓ | 翻阅历史命令 |
| Ctrl-A | 跳到行首 |
| Ctrl-E | 跳到行尾 |
| Ctrl-U | 删除整行 |
| Ctrl-W | 删除单词 |
| Ctrl-C | 中断输入 |
| Ctrl-D | 退出程序 |

## 📁 配置文件
- **历史**: `~/.claw-code-rust/history`
- **API**: `~/.claw-code-rust/config.json`
